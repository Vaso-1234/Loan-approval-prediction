# Loan Approval Prediction

A machine learning project to predict whether a loan application will be approved or not based on applicant details.

## Features
- Data preprocessing and cleaning
- Model training for loan approval prediction
- Prediction using saved model and scaler
- Evaluation report generation

## Files
- `app.py` – main app / prediction interface
- `train_model.py` – model training script
- `preprocess.py` – data preprocessing
- `loan_data.csv` – dataset
- `model.pkl` – trained model
- `scaler.pkl` – saved scaler
- `evaluation_report.json` – model evaluation results

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
